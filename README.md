# Meteor

日本語タイトルは「夜の砂漠に星が降る（仮題）」とする。

# 概要

　砂漠を舞台にしたノベルゲームの開発プロジェクト。  
　アバター（立ち絵）と背景を3Dモデルで表現したノベルゲームで、3Dであることを生かしてカメラやアバターがグリグリ動く。
